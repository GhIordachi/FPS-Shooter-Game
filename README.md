# FPS-Shooter-Game
 
## Version
- **Unity Version:** 2022.3.13f1

## Controls
- **WASD:** Move
- **Hold Shift:** Run
- **Mouse:** Control camera
- **Q:** Quick turn
- **Space:** Jump
- **Right Click:** Aim
- **Left Click:** Shoot
- **R:** Reload
- **E:** Interact (look for checkpoints, an ammo box on a barrel with a flag)
- **X:** Consume (health kits restore life)
- **1:** Equip rifle
- **2:** Equip shotgun
- **3:** Equip revolver
- **Escape:** Pause menu
